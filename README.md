# Formality
A dummy project to build web forms and stuff.

#### Usage:
Run this command:
<br>
`docker-compose up -d --build`
<br>
or those:
<br>
`dotnet run` in `src/Formality.Api`
<br>
`npm run serve` in `src/Formality.Client`

Then you can proceed to http://localhost:8080/.
